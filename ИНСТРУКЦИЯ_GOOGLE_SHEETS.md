# Инструкция по подключению формы Contact к Google Таблице

## Шаг 1: Подготовка Google Таблицы

1. Откройте вашу Google Таблицу

2. Убедитесь, что в таблице есть заголовки столбцов (в первой строке):
   - **A1**: Дата и время
   - **B1**: Имя
   - **C1**: Фамилия
   - **D1**: Email
   - **E1**: Сообщение

   Если заголовков нет, добавьте их вручную.

## Шаг 2: Создание Google Apps Script

1. В открытой Google Таблице нажмите в меню **"Расширения"** → **"Apps Script"**

2. Откроется редактор кода. Удалите весь код по умолчанию (function myFunction() {...})

3. Откройте файл `google-apps-script.js` в вашем проекте

4. Скопируйте весь код из этого файла и вставьте в редактор Apps Script

5. Нажмите на иконку дискеты 💾 или **Ctrl+S** (Cmd+S на Mac) для сохранения

6. Дайте проекту название, например: "Contact Form Handler"

## Шаг 3: Развертывание веб-приложения

1. В редакторе Apps Script нажмите кнопку **"Развернуть"** → **"Новое развертывание"**

2. Нажмите на иконку шестеренки ⚙️ рядом с "Выберите тип"

3. Выберите **"Веб-приложение"**

4. Заполните настройки:
   - **Описание**: "Contact Form Handler" (или любое другое)
   - **Выполнять от имени**: **"Я"** (ваш аккаунт)
   - **У кого есть доступ**: **"Все"** (важно!)

5. Нажмите **"Развернуть"**

6. Возможно, потребуется авторизация:
   - Нажмите **"Авторизовать доступ"**
   - Выберите ваш Google аккаунт
   - Нажмите **"Дополнительно"** → **"Перейти на страницу..."** (если появится предупреждение)
   - Нажмите **"Разрешить"**

7. **ВАЖНО**: Скопируйте **URL веб-приложения** (он выглядит примерно так):
   ```
   https://script.google.com/macros/s/AKfycby.../exec
   ```
   Этот URL понадобится для следующего шага!

## Шаг 4: Обновление кода сайта

1. Откройте файл `script.js` в вашем проекте

2. Найдите строку (около строки 395):
   ```javascript
   const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';
   ```

3. Замените `'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE'` на URL, который вы скопировали на шаге 3.7

   Должно получиться примерно так:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycby.../exec';
   ```

4. Сохраните файл

## Шаг 5: Тестирование

1. Откройте ваш сайт `artsite.html` в браузере

2. Нажмите на ссылку **"Contact"** в меню

3. Заполните форму тестовыми данными:
   - Имя: Тест
   - Фамилия: Тестов
   - Email: test@test.com
   - Сообщение: Тестовое сообщение

4. Нажмите **"Submit"**

5. Если всё работает правильно:
   - Появится сообщение "Спасибо..."
   - В вашей Google Таблице появится новая строка с данными

## Возможные проблемы и решения

### Проблема: Данные не попадают в таблицу

**Решение 1**: Проверьте URL веб-приложения
- Убедитесь, что вы скопировали полный URL
- URL должен заканчиваться на `/exec`

**Решение 2**: Проверьте настройки доступа
- Вернитесь в Apps Script
- Нажмите "Развернуть" → "Управление развертываниями"
- Убедитесь, что "У кого есть доступ" = "Все"

**Решение 3**: Проверьте ID таблицы в скрипте
- В файле `google-apps-script.js` проверьте, что вы заменили YOUR_SPREADSHEET_ID на ваш реальный ID:
  ```javascript
  var SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID'; // ЗАМЕНИТЕ на ID вашей таблицы
  ```
  ID таблицы можно найти в URL: `https://docs.google.com/spreadsheets/d/[ВАШ_ID]/edit`

### Проблема: Ошибка авторизации

**Решение**:
- Повторите процесс авторизации
- Убедитесь, что вы авторизовались под тем же аккаунтом, которому принадлежит таблица

## Дополнительно: Форматирование таблицы

Вы можете настроить форматирование столбцов в Google Таблице:

1. **Столбец A (Дата и время)**:
   - Выделите столбец A
   - Формат → Число → Дата и время

2. **Столбец E (Сообщение)**:
   - Выделите столбец E
   - Увеличьте ширину столбца для удобства чтения длинных сообщений
   - Формат → Перенос текста → Переносить

## Примечания

- Данные добавляются в конец таблицы автоматически
- Каждая отправка формы создает новую строку
- Дата и время записываются автоматически при отправке
- Все данные сохраняются постоянно в вашей Google Таблице
