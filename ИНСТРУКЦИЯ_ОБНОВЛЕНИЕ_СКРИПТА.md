# Инструкция по обновлению Google Apps Script для форм заказов

## Что изменилось?

Теперь Google Apps Script обрабатывает **четыре типа форм**:
1. **Контактная форма** - сохраняется в таблицу "Contacts" (БЕЗ отправки email)
2. **Форма заказа** - сохраняется в таблицу "Orders"
3. **Форма комиссий** - сохраняется в таблицу "Commissions"
4. **Форма пожертвований** - сохраняется в таблицу "Donations"

## Шаг 1: Обновите код в Google Apps Script

1. Откройте вашу Google Таблицу

2. Перейдите в **Расширения** → **Apps Script**

3. **ПОЛНОСТЬЮ УДАЛИТЕ** весь текущий код

4. Откройте файл `google-apps-script.js` в папке вашего проекта

5. **Скопируйте ВЕСЬ КОД** из файла `google-apps-script.js` и вставьте в редактор Apps Script

6. Проверьте настройки в начале кода (строка 16):
   ```javascript
   var SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID'; // ЗАМЕНИТЕ на ID вашей таблицы
   ```
   ID таблицы можно найти в URL: `https://docs.google.com/spreadsheets/d/[ВАШ_ID]/edit`

7. Сохраните: **Ctrl+S** (или Cmd+S на Mac)

## Шаг 2: Обновите развертывание

**ВАЖНО:** Нужно создать НОВОЕ развертывание, чтобы изменения вступили в силу!

1. Нажмите **"Развернуть"** → **"Управление развертываниями"**

2. Найдите ваше текущее развертывание

3. Нажмите на иконку карандаша ✏️ (Редактировать)

4. В поле **"Новая версия"** выберите **"Новая версия"**

5. Добавьте описание версии: "Добавлены формы заказов и комиссий"

6. Нажмите **"Развернуть"**

7. **URL останется прежним** (это важно, потому что он уже используется в коде сайта)

## Шаг 3: Проверьте таблицу

После обновления скрипта в вашей Google Таблице автоматически будут созданы новые листы:
- **Contacts** - для контактных форм
- **Orders** - для заказов
- **Commissions** - для запросов комиссий
- **Donations** - для пожертвований

Листы создадутся автоматически при первой отправке соответствующей формы.

## Как работают email уведомления?

**Email уведомления НЕ отправляются автоматически.** Все данные только сохраняются в Google Таблицу.

Чтобы получать уведомления на email:
1. Откройте вашу Google Таблицу
2. Перейдите в "Инструменты" → "Правила уведомлений"
3. Нажмите "Создать правило"
4. Выберите:
   - "Уведомлять меня при изменении листа"
   - "Немедленно"
5. Сохраните

Теперь вы будете получать email каждый раз, когда в таблицу добавится новая строка.

### Для связи с клиентами:

На сайте есть кликабельная ссылка на вашу почту - при клике на неё откроется почтовый клиент для отправки письма напрямую. Не забудьте заменить `your-email@example.com` на ваш реальный email в файлах `guliegalerie.html` и `script.js`.

## Возможные проблемы

### Проблема: Email уведомления не приходят

**Решение:**
Email уведомления не настроены в скрипте. Чтобы получать уведомления:
1. Настройте правила уведомлений в Google Таблице (см. раздел выше)
2. Или проверяйте таблицу вручную периодически

### Проблема: Данные не сохраняются в таблицу

**Решение:**
1. Проверьте, что вы создали новое развертывание (Шаг 2)
2. Проверьте ID таблицы в коде (строка 16) - замените YOUR_SPREADSHEET_ID на ваш реальный ID
3. Убедитесь, что таблица не защищена от редактирования

### Проблема: Ошибка при отправке формы

**Решение:**
1. Откройте консоль разработчика в браузере (F12)
2. Попробуйте отправить форму снова
3. Посмотрите на ошибки в консоли
4. Убедитесь, что URL скрипта в script.js правильный

## Примечания

- Все данные автоматически сохраняются только в Google Таблицу
- Email уведомления НЕ отправляются автоматически скриптом
- Листы создаются автоматически при первой отправке формы
- Запросы на комиссии сохраняются в отдельный лист "Commissions"
- Пожертвования записываются в лист "Donations" (фактической оплаты не происходит - это только запись намерения)
- Для прямой связи клиенты могут использовать email ссылку на сайте
